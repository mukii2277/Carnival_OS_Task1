# Carnival OS Task 1

Welcome to the Carnival OS Task 1 repository! This project is a challenge designed to teach the basics of Open Source contributions, specifically focusing on fixing documentation and submitting Pull Requests.

## Overview

The goal of this task is to take a poorly formatted README file and transform it into a clean, professional, and well-structured document using proper Markdown syntax.
